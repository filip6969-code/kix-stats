# KIX Stats — Upute za postavljanje

## Što je ovo?
Besplatna web aplikacija za praćenje košarkaških statistika. Radi offline, sprema podatke u browser, može se instalirati kao desktop app.

## Postavljanje (5 koraka, ~10 minuta)

### 1. Napravi GitHub account
Idi na [github.com](https://github.com) i registriraj se (besplatno).

### 2. Napravi novi repozitorij
- Klikni zeleni gumb **"New"** ili **"+"** > **"New repository"**
- Naziv: `kix-stats`
- Ostavi sve ostalo kao default
- Klikni **"Create repository"**

### 3. Uploadaj datoteke
- Klikni **"uploading an existing file"**
- Prevuci sve datoteke iz ovog foldera (index.html, manifest.json, sw.js, icon.svg, i foldere css/ i js/)
- Klikni **"Commit changes"**

### 4. Uključi GitHub Pages
- Idi na **Settings** > **Pages** (lijevi meni)
- Pod "Source" odaberi **"Deploy from a branch"**
- Branch: **main**, folder: **/ (root)**
- Klikni **Save**

### 5. Gotovo!
Za ~2 minute aplikacija je dostupna na:
`https://TVOJE-IME.github.io/kix-stats/`

## Instalacija kao desktop app (Chrome/Edge)
1. Otvori link u Chrome ili Edge
2. U adresnoj traci desno vidiš ikonu za instalaciju (računalo s +)
3. Klikni i odaberi "Install"
4. Aplikacija se otvori kao zaseban prozor — kao nativna app

## Korištenje
- **Utakmica** — praćenje live statistika
- **Povijest** — sve završene utakmice, izvoz u PDF i Excel (.csv)
- Podaci se automatski spremaju dok radiš (draft)
- Klikni "Završi utakmicu" kad je gotovo — spremi se u povijest

## Dijeljenje s prijateljem
Jednostavno pošalji link — isti link, svako ima svoje lokalne podatke u svom browseru.
